# 节假日爬虫

爬虫示例，从Investing.com的节假日日历中收集节假日数据

## 安装

请按照以下步骤在本地设置项目：

### 克隆仓库

   ```bash
   git clone https://github.com/QinjianTest/crawler.git
  ```

### 创建虚拟环境 
```bash
python -m venv .venv
```

### 激活虚拟环境 
#### 在 Unix 或 MacOS 上
```bash
source .venv/bin/activate
```

#### 在 Windows 上
```bash
.venv\Scripts\activate
```

## 安装依赖项 
```bash
pip install -r requirements.txt
```

## 使用说明
运行以下命令以执行爬虫并收集节假日数据：
```bash
python scraper.py
```
